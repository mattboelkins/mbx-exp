Suppose that an object moving along an axis has instantaneous velocity \(v(t) = 2t-3\) and its position at time \(t=1\) is \(s(1) = -1\). What is the position of the object at \(t = 5\)?
Hint Don't forget that the velocity function is the derivative of position. Answer\(s(5) = 11\)Solution
Since \(v(t) = 2t-3\), it follows that \(s(t) = t^2 - 3t + C\). Moreover, because \(s(1)=-1\), we see that \(-1 = 1^2 - 3 \cdot 1 + C\), co \(C = 1\). Thus, \(s(t) = t^2 - 3t + 1\), and \(s(5) = 11\).
